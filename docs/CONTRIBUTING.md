# Contributing

Thanks for (considering) helping.

## How to contribute
- Check Issues for "good first issue" and "help wanted"
- Open an Issue before big changes
- Keep PRs small and focused

## Local setup
Setup instructions will be added once the tech stack is chosen.

## Code style
- Prefer clear names and small functions
- Add tests where it matters
- Update docs if behavior changes

## Security and privacy
If you find a security issue, follow SECURITY.md.
